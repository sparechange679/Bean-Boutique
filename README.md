# Bean Boutique Coffee Shop Website
## Front End Web Development Assignment

### Project Overview
This is a prototype website for Bean Boutique, a boutique coffee shop expanding its online presence. The website is built using only HTML5 and CSS3, with no JavaScript, as per assignment requirements.

### File Structure
```
bean-boutique-website/
├── index.html                  (Home page with slideshow and modal)
├── coffee-selection.html       (Coffee catalogue with search)
├── brewing-equipment.html      (Equipment listings)
├── shopping-cart.html          (Shopping cart page)
├── events.html                 (Events and registration form)
├── special-offers.html         (Subscriptions and offers)
└── styles.css                  (Main stylesheet)
```

### Pages Included (6 Pages)

1. **index.html** - Home Page
   - Navigation bar with hover effects
   - Header section
   - CSS-only slideshow (3 slides, auto-rotating)
   - Modal popup for email signup
   - Featured products section
   - Information boxes with hover effects
   - Social media integration placeholder
   - Interactive map placeholder
   - Security features information
   - Footer

2. **coffee-selection.html** - Coffee Selection
   - Animated search box
   - Product catalogue (12 coffee varieties)
   - Single-origin and blend sections
   - Tasting notes and brewing recommendations
   - Responsive product grid
   - Educational content about coffee

3. **brewing-equipment.html** - Brewing Equipment
   - Product catalogue by category
   - Detailed descriptions and usage tips
   - Espresso machines, brewers, grinders, accessories
   - Brewing methods guide
   - Hover effects on product cards

4. **shopping-cart.html** - Shopping Cart
   - Visual representation of cart items
   - Sample products in cart
   - Cart totals and calculations
   - Recommended products section
   - Shipping and returns information
   - Save cart form

5. **events.html** - Events & Workshops
   - Event listings with dates and details
   - Responsive registration form with:
     - First name, last name (required)
     - Email address (required)
     - Phone, event selection, guest count
     - Experience level, special requests
   - Workshop information
   - Private events section

6. **special-offers.html** - Special Offers & Subscriptions
   - Current promotions
   - Three subscription tiers
   - Subscription benefits
   - Custom subscription form
   - Gift subscription options
   - Loyalty rewards program

### CSS Features Implemented

#### Responsive Design
- Mobile-first approach
- Media queries for tablets and mobile devices
- Flexible grid layouts
- Responsive navigation

#### Visual Effects
- Hover effects on navigation, buttons, and cards
- CSS animations (slideshow, fade-ins, slide-ins)
- Box shadows and border-radius
- Gradient backgrounds
- Transform effects (translateY, scale)
- Transition effects (all interactive elements)

#### Layout Techniques
- CSS Grid for product catalogues and info boxes
- Flexbox for navigation and card layouts
- Sticky navigation
- Fixed positioning for modal overlay

#### Typography & Colors
- Color scheme: Brown/coffee tones (#6f4e37, #8b6f47, #4a3f35)
- Serif fonts for elegance
- Proper hierarchy with heading sizes
- Text shadows for headers

#### Special CSS-Only Features
1. **Slideshow** - Uses CSS animations and keyframes
2. **Modal Popup** - Uses :target pseudo-class
3. **Animated Search** - Transform and box-shadow on focus
4. **Product Cards** - Transform on hover
5. **Form Styling** - Custom styled form elements

### Plugins/Widgets Mentioned (3+)

1. **Social Media Feed** - Placeholder for Instagram/Facebook integration
2. **Interactive Map** - Location information for shop and events
3. **Security Features** - SSL certification and PCI compliance indicators

Note: These are represented as content placeholders since actual plugin integration would require JavaScript.

- Tested on modern browsers (Chrome, Firefox, Safari, Edge)
